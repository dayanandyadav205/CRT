Syllabus CS-310 CRT (Class Room Training)

Week 1: HTML (Days 1-5)

Day 1: Introduction to HTML, basic structure, tags, elements, and attributes.
Day 2: Headings, paragraphs, lists, images, and links.
Day 3: Tables, forms, and input types.
Day 4: Semantic HTML: header, footer, nav, article, section, etc.
Day 5: HTML5 elements: video, audio, iframe, and mini portfolio page.

Week 2: CSS (Days 6-10)

Day 6: CSS syntax, selectors, colors, and font styling.
Day 7: Box model, margin, padding, borders, display.
Day 8: Positioning (static, relative, absolute), float, flexbox.
Day 9: CSS Grid and responsive layouts using media queries.
Day 10: CSS transitions and mini landing page project.
Week 3: JavaScript Basics (Days 11-17)
Day 11: Intro to JS, variables (let/const), and data types.
Day 12: Operators and conditional statements.
Day 13: Loops: for, while, do-while.
Day 14: Nested loops and break/continue statements.
Day 15: Mini practice session on conditions and loops.
Day 16: Basic debugging with console and dev tools.
Day 17: Overview project: simple quiz or logic task.
Week 4: Arrays & Functions (Days 18-24)
Day 18: Intro to arrays, creating, accessing, updating elements.
Day 19: Array methods: push, pop, shift, unshift, splice, slice.
Day 20: Looping through arrays: for, forEach, map.
Day 21: Functions: declaration, expression, arrow functions.
Day 22: Function parameters, return, and scope.
Day 23: Callback functions, nested functions.
Day 24: Practice task using functions and arrays.
Week 5: DOM Manipulation (Days 25-31)
Day 25: Understanding DOM structure and selectors.
Day 26: getElementById, getElementsByClassName, querySelector.
Day 27: Changing element content and attributes dynamically.
Day 28: Creating, appending, and removing elements.
Day 29: Event handling: click, input, submit, mouse events.
Day 30: Form validation and DOM-based tasks.
Day 31: Mini DOM project: To-Do List or Quiz App.
Week 6: Advanced JS (Days 32-38)
Day 32: String and number methods.
Day 33: Object manipulation and nested data.
Day 34: ES6: destructuring, spread, rest, template literals.
Day 35: Asynchronous JS: setTimeout, setInterval.
Day 36: Promises and async/await.
Day 37: LocalStorage and sessionStorage.
Day 38: Practice with async, storage, and ES6 features.
Week 7: jQuery (Days 39-43)
Day 39: jQuery intro, CDN setup, basic selectors.
Day 40: DOM manipulation and CSS class handling.
Day 41: jQuery events and animations.
Day 42: AJAX intro with jQuery.
Day 43: Mini project using jQuery and AJAX.
Week 8: Final Project Using Fetch API (Days 44-48)
Day 44: Choose API (weather/news/todo), setup HTML/CSS.
Day 45: Fetch API GET: Displaying data.
Day 46: Fetch API POST: Form submission.
Day 47: Dynamic DOM updates, loading states.
Day 48: Testing, error handling, and final polish.

Week 9: Project Finalization & Review (Days 49-52)

Day 49: Full review of key JS concepts (arrays, DOM, async).
Day 50: Bug fixing and improvements in the project.
Day 51: Add comments, responsive design to the project.
Day 52: Final project submission and live deployment

Paragraph Formatting

Bold or Strong Text

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Incidunt natus quas aliquam, nemo itaque magni possimus eos rerum qui aspernatur explicabo. Debitis eos iste, velit nobis fugit similique harum dolore.

Italic or Emphasis Text

Lorem ipsum dolor sit amet consectetur adipisicing elit. Quidem eligendi rerum aspernatur distinctio repellat? Sit, totam, quaerat odio, dignissimos provident necessitatibus vel deserunt dolor ex nisi eaque atque autem voluptatem.

Highlighted or Mark Text

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Quis architecto deleniti unde aliquid veniam perferendis provident ratione cum aperiam rerum. Eos quam cupiditate nesciunt quis aut autem officia dolorum vero?
