Intro to arrays

Definition 1: An Array is an object type designed for storing data.
Definition 2: An array is a data structure that stores a collection of items, usually of the same data type, in a single variable at contiguous memory locations (means they are next to each other in sequence without any gaps).

Key characteristics of JavaScript arrays are:

Elements: An array is a list of values, known as elements.

Ordered: Array elements are ordered based on their index.

Zero indexed: The first element is at index 0, the second at index 1, and so on.

Dynamic size: Arrays can grow or shrink as elements are added or removed.

Heterogeneous: Arrays can store elements of different data types (numbers, strings, objects).
